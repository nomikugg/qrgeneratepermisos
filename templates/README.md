# Plantilla PDF Fija

Coloca aqui la plantilla que quieres usar para regenerar permisos desde `/buscar` sin subir archivo cada vez.

Nombre sugerido por defecto:
- `permiso-base.pdf`

Opcional:
- Puedes usar una ruta diferente configurando la variable de entorno `PDF_TEMPLATE_PATH`.
- En Vercel, guarda el archivo dentro del repo y luego haz deploy para que quede disponible en lectura.
